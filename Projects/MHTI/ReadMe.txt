Um das Projekt zu öffnen: 
1. Öffnen Sie bitte Visual Studio Code
2. Öffnen Sie die Datei "index.html"
3. Klicken Sie die rechte Maustaste und wählen bitte "Open with Live Server" 
	(Falls Sie die Extension Live Server nicht haben, 
	 laden Sie sich diese bitte vorher unter den Extensions in Visual Studio 
         Code runter.)
4. Nun können Sie sich die Seite anschauen.

Getestet wurde in Firefox, deshalb empfehle ich diesen Browser zur Ausführung zu nutzen.
